## 📌 About the Project

**SmartStudy Buddy** is an intelligent web-based study assistant designed to make learning more efficient, personalized, and interactive. Whether you're preparing for exams, trying to understand complex topics, or looking for reliable study resources, SmartStudy Buddy acts as your always-available academic companion.

Built with the modern learner in mind, this project bridges the gap between traditional studying and AI-enhanced education — giving students instant access to knowledge, summaries, quizzes, and real-time web information.

---

## 🎯 Purpose

The primary goal of SmartStudy Buddy is to:

- **Reduce study friction** — students shouldn't waste hours searching for quality resources
- **Personalize the learning experience** — adapt responses based on the topic, difficulty, and student needs
- **Make studying interactive** — through AI chat, quizzes, and smart summaries
- **Provide up-to-date information** — using AI-based web search to fetch real-time, relevant content beyond static training data

---

## ✨ Key Features

- 🤖 **AI Study Assistant** — Chat with an intelligent assistant that explains concepts clearly and concisely
- 🌐 **AI-Based Web Search** — Real-time web search capability powered by AI, allowing the assistant to fetch current and relevant study material from across the internet
- 📝 **Smart Summaries** — Paste any text or topic and get an instant, easy-to-understand summary
- 🧠 **Quiz Generator** — Automatically generate practice quizzes from any subject or uploaded content
- 📂 **Subject-Based Organization** — Organize your study sessions by subject or topic
- 📱 **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices

---

## 🌐 AI-Based Web Search Capabilities

One of the standout features of SmartStudy Buddy is its **AI-powered web search integration**.

Unlike traditional chatbots limited to pre-trained data, SmartStudy Buddy can:

- 🔍 Search the web in real-time to answer questions with the latest information
- 📰 Pull recent articles, research, and explanations relevant to your study topic
- 🧾 Summarize web results into digestible, student-friendly answers
- 🔗 Cite sources so students can explore further

This means your study sessions are never blocked by outdated information — the AI searches, filters, and delivers exactly what you need, when you need it.

## 🚀 Getting Started

![Project Outlook](https://i.ibb.co/mr5NNSHb/Screenshot-2026-03-12-001235.png)

### Prerequisites

- Node.js >= 18
- npm or yarn
- API keys for your AI provider

### Installation
```bash
git clone https://github.com/soloStack-Dev/SmartStudy_buddy__.git
cd SmartStudy_buddy__
cd demo
mvn spring-boot:run
```

### Environment Variables

Create a `.env` file in the root directory:
```env
AI_API_KEY=your_api_key_here
SEARCH_API_KEY=tavily_search_api_key
server.PORT=8080
```

### Run Locally
```bash
mvn spring-boot:run
```

Open your browser at `http://localhost:8080`
